# taxi marketing Portfolio

![Portfolio Website](https://i.ibb.co/WgPMpts/image.png)

A modern, responsive personal portfolio website built to showcase projects, skills, accomplishments, and professional experience.

## ✨ Features
- **Hero Section:** Introduction with background animation.
- **Projects Showcase:** Display completed projects with detailed descriptions.
- **Technologies:** Highlight technical skills and tools used.
- **Timeline:** Professional journey and experience timeline.
- **Accomplishments:** Display professional milestones and achievements.

## 🛠️ Tech Stack
- **Framework:** [Next.js](https://nextjs.org/)
- **JS Library:** [React](https://reactjs.org/)
- **Styling:** [styled-components](https://styled-components.com/)
- **Icons:** [react-icons](https://react-icons.github.io/react-icons/)

## 🚀 Getting Started

Follow these steps to run the project locally.

### Prerequisites
Make sure you have Node.js and npm (or yarn) installed on your machine.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/mohamedhamdy9966/frameportfolio.git
   ```
2. Navigate to the project directory:
   ```bash
   cd frameportfolio
   ```
3. Install dependencies:
   ```bash
   npm install
   # or
   yarn install
   ```

### Running the App

Start the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
